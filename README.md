# Prueba-1
#Oskar y Nin es un juego cooperativo de terror psicológico donde dos jugadores deben trabajar juntos para sobrevivir y escapar de un entorno oscuro y peligroso. Ambos comparten una misma vista panorámica, lo que los obliga a mantenerse cerca y coordinarse constantemente. Durante la aventura enfrentarán alucinaciones, copias espectrales y fenómenos extraños mientras resuelven puzles y buscan salir con vida.
